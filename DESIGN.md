# Water Polo Champions League UI

## Layout
- Sticky top nav with global filters (season, stage, search) and freshness pill.
- Max width 1280px, generous card spacing, neutral background with subtle gradients.

## Visual language
- Cards-first layout with soft borders, white glass panels, and restrained accent colors.
- Primary accent: teal (#0ea5a3) for wins/positive deltas.
- Secondary accent: coral (#f97316) for negative deltas.

## Chart intent
- Line charts: rolling form and trends.
- Radar: team profile across core efficiency dimensions.
- Scatter: efficiency vs volume (future).
- Distribution: league snapshot.

## UX conventions
- Filters are URL-driven for shareable deep links.
- Loading states use skeleton blocks.
- Empty states explain missing data and link to admin scrape.
